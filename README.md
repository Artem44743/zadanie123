# Сервис Анализа и Визуализации Данных

Экзаменационный проект по дисциплине «Разработка программных модулей». Вариант 2 (Python).

## Инструкция по запуску

1. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите веб-интерфейс приложения через Streamlit:
   ```bash
   streamlit run main.py
   ```

3. Запустите модульные тесты для проверки логики:
   ```bash
   pytest tests/
   ```
