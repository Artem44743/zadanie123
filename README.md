# 📊 Модульный сервис анализа и визуализации бизнес-данных

Веб-приложение на фреймворке **Streamlit** для интерактивного анализа, фильтрации и визуализации данных о продажах. Проект разработан в рамках экзамена по дисциплине *«Разработка программных модулей»* (Вариант 2, Python).

---

## 🚀 Ключевой функционал

- **Универсальная загрузка данных**: Работа со встроенной автоматически генерируемой демо-базой данных (30 записей) или загрузка пользовательских файлов формата `.csv`.
- **Интерактивная фильтрация**: Динамический срез данных по категориям товаров «на лету» без перезагрузки страницы.
- **Аналитическая сводка (KPI)**: Расчет ключевых бизнес-метрик (Общая выручка, Количество проданных штук, Средний чек товара).
- **Интерактивные дашборды (Plotly)**:
  - Линейный график динамики выручки во времени с маркерами точек.
  - Круговая диаграмма (Donut chart) распределения долей выручки по категориям.
- **Экспорт результатов**: Возможность выгрузить отфильтрованный срез данных обратно в чистый `.csv`-файл одной кнопкой.
- **Автоматическое тестирование**: Полноценное покрытие бизнес-логики модульными тестами через `pytest`.

## 📂 Архитектура и структура проекта

Проект строго следует принципу разделения ответственности (**Separation of Concerns**). Бизнес-логика полностью изолирована от интерфейса отображения.

```text
project/
│
├── app/
│   ├── __init__.py
│   ├── routes/            # Слой интерфейса (Отображение страниц)
│   │   ├── __init__.py
│   │   ├── home.py        # Страница загрузки и обзора таблиц
│   │   └── analytics.py   # Страница графиков и KPI дашбордов
│   │
│   ├── services/          # Сервисный слой (Бизнес-логика)
│   │   ├── __init__.py
│   │   └── data_manager.py # Фильтрация, расчеты и валидация CSV
│   │
│   └── utils/             # Вспомогательные модули
│       ├── __init__.py
│       └── helpers.py     # Автогенерация демо-данных
│
├── data/
│   └── mock_data.csv      # Локальный файл сгенерированной БД
│
├── tests/                 # Модульное тестирование
│   ├── __init__.py
│   └── test_analytics.py  # Тесты на расчеты и фильтрацию
│
├── main.py                # Главная точка входа приложения
├── requirements.txt       # Зависимости проекта
└── README.md              # Документация проекта
```

## 🛠️ Инструкция по локальному запуску

### 1. Подготовка окружения и установка
Рекомендуется использовать изолированное виртуальное окружение `venv` во избежание конфликтов путей:

```bash
# Создание виртуального окружения
python -m venv venv

# Активация окружения (Windows PowerShell)
.\venv\Scripts\Activate.ps1

# Установка необходимых библиотек
pip install -r requirements.txt
```

### 2. Запуск веб-интерфейса
Для обхода ограничений длины путей Windows (`MAX_PATH`) запуск веб-сервера оптимизирован напрямую через интерпретатор:

```bash
python main.py
```
После запуска откройте в браузере адрес: **`http://127.0.0.1:9999`**

### 3. Запуск модульных тестов
Для проверки корректности математических алгоритмов расчетов выполните команду:
```bash
python -m pytest tests/
```
